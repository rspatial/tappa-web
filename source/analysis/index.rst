Spatial data analysis
=====================

This is the *Python* / *tappa* port of the
`Spatial data analysis <https://rspatial.org/analysis/>`__ section of
*rspatial.org*. All eight chapters are ported; some advanced R-only
workflows (e.g. full `spatstat` census interpolation) are approximated
with PySAL / *pointpats* equivalents.
