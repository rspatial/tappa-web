Spatial Data Science with Python and "tappa"
============================================

These resources teach spatial data analysis and modeling with *Python*. *Python*
is a widely used programming language with a rich scientific stack
(*NumPy*, *pandas*, *matplotlib*, ...). The
`tappa <https://github.com/rspatial/tappa>`__ package brings the *terra* spatial
data model to Python and is used throughout these tutorials.

If you are new to *Python*, we recommend you first work through the
`official Python tutorial <https://docs.python.org/3/tutorial/>`__ and the
`Scientific Python <https://learn.scientific-python.org/development/>`__
introduction before you start here.
